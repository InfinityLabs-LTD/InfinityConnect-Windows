# Авто-обновление приложения

InfinityConnect использует официальный **Tauri Updater** + **GitHub Releases**.
Приложение проверяет обновления по `latest.json` в релизах, скачивает подписанный
установщик, проверяет подпись и ставит его с перезапуском.

Код уже готов (плагины, UI-карточка в «О приложении», проверка/скачивание/прогресс).
Осталась разовая настройка инфраструктуры — её делаешь **ты** (ключи нельзя коммитить).

## 1. Один раз: ключи подписи

Обновления подписываются, чтобы никто не подсунул зловред под видом апдейта.

```powershell
# Генерируем пару ключей (задаст пароль — запомни его).
npm run tauri signer generate -- -w $env:USERPROFILE\.tauri\infinity.key
```

Появятся два файла:
- `infinity.key` — **приватный** ключ. НИКОМУ не давать, НЕ коммитить.
- `infinity.key.pub` — публичный ключ.

Скопируй содержимое `infinity.key.pub` и вставь в `src-tauri/tauri.conf.json`:
```json
"plugins": { "updater": { "pubkey": "СЮДА_СОДЕРЖИМОЕ_infinity.key.pub" } }
```

## 2. Один раз: endpoint

В `src-tauri/tauri.conf.json` замени `OWNER/REPO` на свой GitHub-репозиторий:
```json
"endpoints": ["https://github.com/OWNER/REPO/releases/latest/download/latest.json"]
```

## 3. При каждом релизе

1. Подними версию в `src-tauri/tauri.conf.json` (`"version": "0.2.0"`).
2. Собери подписанный бандл (приватный ключ через переменные окружения):
   ```powershell
   $env:TAURI_SIGNING_PRIVATE_KEY = Get-Content $env:USERPROFILE\.tauri\infinity.key -Raw
   $env:TAURI_SIGNING_PRIVATE_KEY_PASSWORD = "твой_пароль"
   npm run tauri build
   ```
3. В `src-tauri/target/release/bundle/` появятся:
   - установщик `.exe`/`.msi`
   - подписи `.sig`
   - `latest.json` (манифест обновления)
4. Создай **GitHub Release** с тегом версии и залей туда: установщик, `.sig` и `latest.json`.

Готово — у всех пользователей в «О приложении» появится «Доступна версия …».

## Как это выглядит у пользователя

«Настройки → О приложении → Обновление приложения»:
- «Проверить» → сравнивает версию с `latest.json`.
- Если новее — «Установить X.Y.Z» + описание изменений (release notes).
- Прогресс скачивания → авто-установка → перезапуск в новой версии.

## Заметки

- Пока `pubkey`/`endpoint` не настроены (плейсхолдеры), кнопка «Проверить» вернёт
  ошибку — это ожидаемо до шага 1-2.
- Ядра (xray/hysteria/sing-box) обновляются вместе с приложением (они в бандле).
- Подписки серверов обновляются отдельно (авто, раз в 12ч + при старте).
